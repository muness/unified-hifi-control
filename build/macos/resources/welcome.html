Unified Hi-Fi Control

Welcome! This installer will set up Unified Hi-Fi Control on your Mac.

What gets installed:

• /usr/local/bin/unified-hifi-control - The application
• LaunchDaemon - Runs automatically at startup
• Web UI - Available at http://localhost:8088

Features:

• Control Roon, LMS, and HQPlayer from one interface
• MQTT integration for Home Assistant
• Hardware knob support (ESP32)
• Claude AI integration via MCP
